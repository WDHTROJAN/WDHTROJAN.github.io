Semantic analysis for YouTube user comments

Project information

  • Project: Semantic analysis for YouTube user comments
  • Category: Text Analysis
  • Author: Lesley Wu
  • Location: Los Angeles, CA

Project Description

- Analyzed 6 million YouTube user comments in Spark to identify dog/cat owners and discover latent semantic structures.

- Preprocessed data by removing missing values and labeled dog & cat owners and users who donâ€™t have pets.

- Processed comment text by tokenization, stemming, removing stop words and extracted features via RegexTokenizer and Word2Vec in SparkML.

- Train Logistic regression model and random forest model to classify all users, tunned hyper parameters and selected best model based on accuracy(90%) and AUC(96%).

- Extracted features with term frequency-inverse document frequency(TF-IDF) approach and trained unsupervised learning model Latent Dirichlet Allocation(LDA) to obtain important topic for target users.
