Identity Fraud Detection for Credit Card Applications

Project information

  • Project: Identity Fraud Detection for Credit Card Applications
  • Category: Fraud analysis
  • Author: Lesley Wu
  • Location: Los Angeles, CA

Project Description

- Built a real time fraud algorithm to detect identity fraud in 10K+ credit card applications.

- Performed exploratory data analysis and cleaned the dataset by replacing frivolous values with neutral values.

- Created 600 + candidate variables based on the periodicity & speed of applications and performed feature selection using filter and wrapper methods to reduce all variables to 30 expert variables.

- Trained logistic regression and random forest models to predict fraud applications, selected best model based on recall score(60%) and tune model parameters.
