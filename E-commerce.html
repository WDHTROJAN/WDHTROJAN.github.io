E-commerce Fraud Detection and Risk Analysis

Project information

  • Project: E-commerce Fraud Detection and Risk Analysis
  • Category: Fraud analysis
  • Author: Lesley Wu
  • Location: Los Angeles, CA

Project Description

- Built an alert system to prevent fraudulent activities on an e-commerce website by analyzing data of 138K+ transactions.

- Preprocessed raw data by removing duplicates, encoding categorical features and handling imbalanced data by SMOTE.

- Trained logistic regression and random forest models to predict fraud transaction, selected best model based on AUC (80%) & F1 score (70%) and identified top 2 predictive features for fraud activity.

- Converted the model output(class probability) to a key metric of the alert system to classify customers into 3 risk categories.
